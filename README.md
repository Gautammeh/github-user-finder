# 🔍 GitHub User Finder

A simple and responsive React application that allows users to search for any GitHub profile and view their details along with top repositories.

---

## 🚀 Features

* 🔎 Search any GitHub user by username
* 👤 Display user profile (avatar, bio, followers, following, etc.)
* ⭐ Show top 6 repositories sorted by stars
* ⚡ Fast and responsive UI
* ❌ Handles errors (user not found, API limit, etc.)

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS / Tailwind (mention what you used)
* GitHub REST API
